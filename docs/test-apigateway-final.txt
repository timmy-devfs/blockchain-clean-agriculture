============================================================
VI) FLOW TEST CHINH (COPY-PASTE DEMO)
============================================================

-----------------------------
FLOW 1 - Register user demo
-----------------------------
curl -X POST http://localhost:8081/api/auth/register -H "Content-Type: application/json" -d "{\"email\":\"farm1@bicap.io\",\"password\":\"123456\",\"fullName\":\"Nguyen Van A\",\"role\":\"FARM_MANAGER\"}"

Expected: HTTP 201, message Registered successfully.

-----------------------------
FLOW 2 - Login truc tiep Identity
-----------------------------
curl -X POST http://localhost:8081/api/auth/login -H "Content-Type: application/json" -d "{\"email\":\"farm1@bicap.io\",\"password\":\"123456\"}"

Expected: Co accessToken + refreshToken.
Action: Copy accessToken va refreshToken de dung cho flow sau.

-----------------------------
FLOW 3 - Introspect token dung/sai
-----------------------------
3.1 Token dung:
curl -X POST http://localhost:8081/api/auth/introspect -H "Content-Type: application/json" -d "{\"token\":\"ACCESS_TOKEN\"}"

Expected: valid=true, co userId/email/role.

3.2 Token sai:
curl -X POST http://localhost:8081/api/auth/introspect -H "Content-Type: application/json" -d "{\"token\":\"this.is.fake.token\"}"

Expected: valid=false.

-----------------------------
FLOW 4 - Refresh token
-----------------------------
curl -X POST http://localhost:8081/api/auth/refresh-token -H "Content-Type: application/json" -d "{\"refreshToken\":\"REFRESH_TOKEN\"}"

Expected: Tra cap token moi.

-----------------------------
FLOW 5 - Login qua Gateway
-----------------------------
curl -X POST http://localhost:8080/api/auth/login -H "Content-Type: application/json" -d "{\"email\":\"farm1@bicap.io\",\"password\":\"123456\"}"

Expected: Thanh cong giong login truc tiep identity.
Y nghia: Chung minh gateway route dung.

-----------------------------
FLOW 6 - Login qua Nginx (Layer 1)
-----------------------------
curl http://localhost/nginx-health
curl -X POST http://localhost/api/auth/login -H "Content-Type: application/json" -d "{\"email\":\"farm1@bicap.io\",\"password\":\"123456\"}"

Expected:
- nginx-health = 200
- login thanh cong => chain Nginx -> Gateway -> Identity OK

-----------------------------
FLOW 7 - Protected route (khong token / co token)
-----------------------------
7.1 Khong token:
curl -v http://localhost:8080/api/farm/farms

Expected: 401 Unauthorized.

7.2 Co token:
curl -v http://localhost:8080/api/farm/farms -H "Authorization: Bearer ACCESS_TOKEN"

Expected:
- Khong con 401
- Neu farm-service chua san sang co the 503/fallback (van dung y nghia auth da qua)

-----------------------------
FLOW 8 - Redis JWT cache (MISS -> HIT)
-----------------------------
set TOKEN=ACCESS_TOKEN

Lan 1:
curl -v http://localhost:8080/api/farm/farms -H "Authorization: Bearer %TOKEN%"
Expected: Cache MISS (xem log gateway neu can).

Lan 2 (cung token):
curl -v http://localhost:8080/api/farm/farms -H "Authorization: Bearer %TOKEN%"
Expected: Cache HIT.

Kiem tra key cache:
docker exec bicap-redis redis-cli keys "jwt:*"


-----------------------------
FLOW 9 - Rate limiting
-----------------------------
for /L %i in (1,1,110) do curl -s -o NUL -w "%{http_code} " http://localhost:8080/api/gateway/health

Expected:
- Ban dau chu yeu 200
- Sau nguong (~100 req/phut) xuat hien 429 Too Many Requests
