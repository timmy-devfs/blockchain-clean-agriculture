II) KHOI DONG STACK DEMO
========================

Chay theo thu tu:

1) Dung + xoa stack cu:
   docker compose down --volumes --remove-orphans

2) Build + up:

3) Tao Kafka topics:

Expected:
- Container chinh Up/healthy.
- nginx-health tra status UP.
- gateway actuator health la UP.


III) FIX MONGO REPLICA SET (NEU CAN)
====================================

Neu farm/retailer/payment bi loi transaction (Prisma Mongo):

1) Check:
   docker exec bicap-mongo mongosh --quiet --eval "rs.status().ok"

2) Neu chua co rs:
   docker exec bicap-mongo mongosh --quiet --eval "rs.initiate({_id:'rs0', members:[{_id:0, host:'mongo:27017'}]})"

3) Restart cac service lien quan:
   docker compose up -d farm-service retailer-service payment-service iot-service blockchain-service shipping-service

VI) FLOW TEST API GATEWAY (CORE)
================================

FLOW 1 - Login qua Gateway
--------------------------
curl -X POST http://localhost:8080/api/auth/login -H "Content-Type: application/json" -d "{\"email\":\"admin@bicap.io\",\"password\":\"123456\"}"

Expected:
- Tra accessToken + refreshToken.


FLOW 2 - Login qua Nginx (Layer 1 -> Gateway)
---------------------------------------------
curl -X POST http://localhost/api/auth/login -H "Content-Type: application/json" -d "{\"email\":\"farm1@bicap.io\",\"password\":\"123456\"}"

Expected:
- Login thanh cong nhu qua :8080.


FLOW 3 - Introspect token
-------------------------
curl -X POST http://localhost:8080/api/auth/introspect -H "Content-Type: application/json" -d "{\"token\":\"ACCESS_TOKEN\"}"

Expected:
- valid=true + co thong tin user.


FLOW 4 - Protected route (khong token / co token)
-------------------------------------------------
Khong token:
curl -i http://localhost:8080/api/farm/farms

Expected:
- 401 Unauthorized.

Co token:
curl -i http://localhost:8080/api/farm/farms -H "Authorization: Bearer ACCESS_TOKEN"

Expected:
- Khong con 401 (co the 200/503 tuy backend, nhung auth da qua Gateway).

IX) TROUBLESHOOT NHANH
======================

1) Login bi ERR_CONNECTION_REFUSED localhost:8080
- Gateway chua up hoac chua map 8080:8080.
- Chay:
  docker compose up -d --force-recreate api-gateway

2) Login bi CORS
- Chay lai lenh preflight o muc FLOW 7.
- Rebuild gateway neu vua sua config:
  docker compose up -d --build api-gateway

3) UNKNOWN_TOPIC_OR_PARTITION
- make make-topics
- restart service consumer (farm/blockchain/shipping...)

4) Loi transaction Mongo/Prisma
- Bat replica set theo muc III.

5) Nginx 502 hang loat
- Restart nginx:
  docker compose restart nginx


X) STOP DEMO
============

docker compose down
