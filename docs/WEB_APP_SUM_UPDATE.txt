BICAP WEB-APP MIGRATION AUDIT SUMMARY
=====================================

Ngay cap nhat: 2026-05-01
Phan vi: Tong ket migration frontend sau cac phase cleanup va verify.


I) KET QUA MIGRATION CHINH
==========================

1) Unified frontend
- Da chuan hoa ve 1 app duy nhat: `frontend/web-app`.
- Route theo role + public duoc to chuc tren App Router:
  - `/admin/*`, `/farm/*`, `/retailer/*`, `/shipping/*`
  - `/public`, `/articles`, `/products`, `/trace`, `/how-it-works`, `/huong-dan`

2) API gateway convention
- Browser-facing API base URL da thong nhat:
  `NEXT_PUBLIC_API_URL=http://localhost/api`
- Han che hardcode goi truc tiep service port rieng.

3) Shipping sync fallback
- Writer: `app/api/sync-orders/route.ts`
- Reader fallback: `app/internal/shipping-sync/route.ts`
- Admin shipments fallback:
  - neu shipping-service rong/loi -> doc sync data
  - neu khong co data -> hien empty state, khong crash

4) Public pages duplicate cleanup
- Da bo `public-site/app/` duplicate.
- Public route active la `app/(public)/...`.
- Van giu `public-site/data/` theo dung muc tieu.
