# blockchain-clean-agriculture
Giới thiệu dự án  AgriChain là một hệ thống ứng dụng công nghệ Blockchain nhằm nâng cao tính minh bạch, truy xuất nguồn gốc và đảm bảo chất lượng trong sản xuất nông nghiệp sạch. Hệ thống cho phép ghi nhận toàn bộ quá trình từ gieo trồng, chăm sóc, thu hoạch, vận chuyển đến phân phối dưới dạng dữ liệu bất biến trên Blockchain.
